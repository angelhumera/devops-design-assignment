# DevOps Design Assignment

This repository contains the design documentation for a simple 3-tier backend system.

- `backend_design.md` → Backend API architecture (EC2 + Python)
- `frontend_design.md` → Frontend architecture
- `database_design.md` → Database design
- `cicd_design.md` → CI/CD design
- `runbooks/deploy_rollback.md` → Runbooks for deployment and rollback
- `diagram/architecture.drawio` → Architecture diagram
- `diagram/architecture_design.pdf` → Exported PDF of diagram

This is a **design-only submission**, no deployment or coding is required.
